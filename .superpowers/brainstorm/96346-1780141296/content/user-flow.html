How should the platform handle the input collection phase?

After the user confirms their selected modules, they need to provide specific information. How should this work?

A

Single Combined Form

Show all required fields from all selected modules on one page. User fills everything at once and submits.

Pros: Fast for users who have all info ready
Cons: Can be overwhelming if many modules selected
B

Step-by-Step Wizard

One module per screen. User completes Housing fields, clicks Next, then Legal fields, etc. Progress indicator shows completion.

Pros: Less overwhelming, clear progress, easier validation
Cons: More clicks, can't skip around easily
C

Expandable Sections

All modules visible on one page as collapsed sections. User can expand any module, fill it out, collapse it, and move to the next in any order.

Pros: Flexibility, see everything at once, non-linear completion
Cons: Less clear what's required, easy to miss sections
